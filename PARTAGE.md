# 📤 COMMENT PARTAGER LE JEU

## 🎯 3 FAÇONS SIMPLES DE PARTAGER

---

## **OPTION 1️⃣ : PARTAGER EN PYTHON (PLUS FACILE)**

### Fichier à envoyer :
- `simulation.py`

### Vous envoyez :
```
📧 Email
📎 Fichier : simulation.py
💬 Message : "Lance : python simulation.py"
```

### Ils reçoivent et font :
```bash
python simulation.py
```

### ✅ Avantages :
- Aucune installation supplémentaire
- Fonctionne Windows/Mac/Linux
- Petit fichier

### ❌ Inconvénient :
- Besoin de Python
- Pas de graphique (terminal)

---

## **OPTION 2️⃣ : PARTAGER L'APPLICATION WINDOWS (.EXE)**

### Fichier à envoyer :
- `reaction_time.exe`

### Vous compilez d'abord :
```bash
compile.bat
```

### Vous envoyez :
```
📧 Email
📎 Fichier : reaction_time.exe
💬 Message : "Double-clic pour jouer"
```

### Ils reçoivent et font :
```
👆 Double-clic sur reaction_time.exe
🎮 Le jeu s'ouvre !
```

### ✅ Avantages :
- Très professionnel
- Graphique beau
- Aucune installation de leur côté

### ❌ Inconvénient :
- Fichier plus gros (~150 MB)
- Windows seulement
- Demande compilation

---

## **OPTION 3️⃣ : PARTAGER LE DOSSIER COMPLET (PRO)**

### Dossier à créer :
```
Reaction-Time/
├── README_SIMPLE.md
├── GUIDE_FINAL.md
├── simulation.py
├── reaction_time.c
├── Makefile
├── START.bat
└── reaction_time.exe (optionnel, après compilation)
```

### Zipez-le :
```
Reaction-Time.zip
```

### Vous envoyez :
```
📧 Email / Drive / WeTransfer
📎 Fichier : Reaction-Time.zip
💬 Message : "Décompressez et lancez START.bat"
```

### Ils reçoivent et font :
```
1. Décompressez le ZIP
2. Lancez : START.bat
3. Choisissez option 1 ou 2
4. Jouez !
```

### ✅ Avantages :
- Complètement autonome
- Menu facile
- Code source visible
- Instructions incluses

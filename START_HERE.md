## ✅ VÉRIFICATION RAPIDE

Tout fonctionne si :
- ✅ Double-clic sur `PLAY.bat` → Le jeu se lance
- ✅ Vous voyez un menu couleur terminal
- ✅ Code rouge, puis vert s'affichent
- ✅ Vous pouvez appuyer sur ENTRÉE

Si erreur Python → Installez Python de https://python.org

## 💡 TIPS

- Vous pouvez rejouer → appuyez ENTRÉE
- Quitter → appuyez Q
Restart → appuyez R
- Plus vous jouez, plus ça devient difficile
- Score EXCELLENT < 200ms ⚡

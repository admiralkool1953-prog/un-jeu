## 🎯 QU'EST-CE QUE C'EST ?

**Reaction Time** est un jeu qui teste votre vitesse de réaction :

```
🔴 ROUGE = Attendez
🟢 VERT = Appuyez VITE sur ENTRÉE
⏱️ = Votre temps en millisecondes
🏆 = Battez votre record
```

---

## ✅ COMMENT JOUER

1. **Lancez le jeu** :
   ```bash
   python simulation.py
   ```

2. **Suivez les instructions** :
   - L'écran devient ROUGE
   - Attendez le VERT
   - Appuyez rapidement sur ENTRÉE

3. **Voyez votre résultat** :
   - Votre temps s'affiche
   - Battez votre meilleur temps

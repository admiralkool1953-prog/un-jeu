## **OPTION B : VERSION GRAPHIQUE (RAYLIB)**

### **Pour compiler** :

#### **Étape 1 : Installer Raylib**

**Windows** :
1. Télécharger : https://github.com/raysan5/raylib/releases
2. Extraire l'archive
3. Copier dans le dossier `raylib/` :
   - `raylib.h` → `raylib/include/`
   - `raylib.a` → `raylib/lib/`

#### **Étape 2 : Compiler le jeu**

```bash
cd "c:\Users\HP\Desktop\mini projet c"
compile.bat
```

#### **Étape 3 : Jouer**

```bash
reaction_time.exe
```

### **Résultat** :
Une fenêtre graphique Raylib s'ouvre avec :
- Fond ROUGE/VERT coloré
- Texte centré et animé
- Professionnellement beau
